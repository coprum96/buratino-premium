## 1️⃣ Система сбора данных для исследования

### Новые файлы
- **`src/utils/sessionAnalytics.ts`** - Основная система аналитики
- **`src/components/DataExportPanel.tsx`** - Панель экспорта данных для исследователей
- **`RESEARCH_DATA_GUIDE.md`** - Подробное руководство по использованию

### Что собирается

#### Игровые метрики
- Уникальный ID каждой сессии
- Временные метки начала и конца игры
- Общее время игры
- Время на каждый уровень
- Пройденные уровни
- Финальные статы (монеты, мудрость)
- Полученные достижения

#### Ответы и выборы
- **Викторины**: все ответы с правильностью
- **Тесты**: детальные результаты (scenario, complex, post)
- **Диалоги**: все выборы с влиянием на статы
- **Материалы**: просмотренные образовательные материалы и время просмотра

### Как использовать

#### Доступ к данным
- Нажмите **`Ctrl+Shift+D`** (Windows/Linux) или **`⌘+Shift+D`** (Mac) на любом экране
- Откроется панель исследователя с статистикой

#### Экспорт данных
1. Откройте панель (Ctrl+Shift+D)
2. Просмотрите статистику
3. Нажмите "Экспортировать данные (JSON)"
4. Файл скачается автоматически

#### Формат данных
- JSON файл с полной структурой всех сессий
- Агрегированная статистика
- Готов для анализа в Python, R, Excel

### Технические детали

#### Интеграция в компоненты
- **StartScreen**: инициализация новой сессии
- **DialogueScene**: отслеживание выборов и начала уровней
- **QuizScreen**: запись ответов на вопросы
- **ScenarioTest**: запись результатов сценарных тестов
- **PostTest**: запись финального теста
- **EndingScreen**: завершение сессии и финальные метрики
- **App.tsx**: глобальный доступ к панели данных

#### Хранение
- Локально в браузере (localStorage)
- Без отправки на сервер (приватность)
- Автоматическое сохранение после каждого действия

## 🚀 Как протестировать

### 1. Запустите приложение
```bash
npm run dev
```

### 2. Проверьте лендинг
- Откройте http://localhost:5173
- Убедитесь что:
  - ✅ Виден бейдж СПбГУ сверху
  - ✅ Нет счетчика жертв
  - ✅ Кнопка "Начать приключение" большая

### 3. Начните игру
- Нажмите "Начать приключение"
- Сессия автоматически начнет записываться

### 4. Проверьте сбор данных
- Пройдите несколько уровней
- Ответьте на вопросы викторины
- Нажмите `Ctrl+Shift+D` (или `⌘+Shift+D`)
- Проверьте статистику в панели

### 5. Экспортируйте данные
- В панели нажмите "Экспортировать данные"
- Откройте скачанный JSON файл
- Убедитесь что данные полные

### 6. Проверьте счетчик прогресса
- В правом верхнем углу должно быть: `X/14`
- После прохождения уровня число должно увеличиваться
