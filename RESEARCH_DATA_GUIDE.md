# 📊 Руководство по сбору данных для исследования

## Обзор

Игра "Золотой Детектор" теперь включает полноценную систему сбора аналитических данных для исследовательских целей. Все данные собираются локально в браузере игрока и могут быть экспортированы для анализа.

## 🎯 Что собирается

### 1. Игровые сессии
- **ID сессии** - уникальный идентификатор
- **Временные метки** - начало и конец сессии
- **ID пользователя** (опционально) - для идентификации участников
- **Общее время игры** - в секундах

### 2. Прогресс игрока
- Пройденные уровни
- Финальные монеты
- Финальная мудрость (%)
- Полученные достижения

### 3. Ответы на викторины
- ID уровня
- Индекс вопроса
- Текст вопроса
- Выбранный ответ
- Правильность ответа
- Временная метка

### 4. Результаты тестов
- Тип теста (scenario/complex/post)
- Общий балл
- Детальные ответы на каждый вопрос
- Временная метка

### 5. Выборы в диалогах
- ID уровня
- Индекс диалога
- Имя персонажа
- Текст выбора
- Изменение мудрости
- Изменение монет
- Временная метка

### 6. Временные метрики
- Время на каждый уровень (в секундах)
- Просмотры образовательных материалов
- Длительность просмотра материалов

## 📊 Анализ данных

### Python пример для анализа

```python
import json
import pandas as pd

# Загрузка данных
with open('buratino-research-data-2024-01-15.json', 'r', encoding='utf-8') as f:
    data = json.load(f)

# Конвертация в DataFrame
sessions_df = pd.DataFrame(data['sessions'])

# Основная статистика
print(f"Всего сессий: {len(sessions_df)}")
print(f"Средняя длительность: {sessions_df['totalPlayTime'].mean():.0f} сек")
print(f"Средняя мудрость: {sessions_df['finalWisdom'].mean():.1f}%")

# Анализ ответов на викторины
all_quiz_answers = []
for session in data['sessions']:
    all_quiz_answers.extend(session['quizAnswers'])

quiz_df = pd.DataFrame(all_quiz_answers)
accuracy = quiz_df['isCorrect'].mean() * 100
print(f"Точность ответов: {accuracy:.1f}%")

# Анализ по уровням
level_times = {}
for session in data['sessions']:
    for level_id, time in session['timePerLevel'].items():
        if level_id not in level_times:
            level_times[level_id] = []
        level_times[level_id].append(time)

# Средние времена по уровням
for level_id, times in sorted(level_times.items()):
    avg_time = sum(times) / len(times)
    print(f"Уровень {level_id}: {avg_time:.0f} сек")
```

## 🛠️ Техническая информация

### Хранилище
- **Технология**: localStorage браузера
- **Ключи**:
  - `currentSession` - текущая активная сессия
  - `sessionsArchive` - все завершенные сессии

### Ограничения
- localStorage обычно ограничен 5-10 МБ
- Примерно 500-1000 сессий в зависимости от детализации
- Данные привязаны к домену и браузеру

### Очистка данных
⚠️ **Осторожно**: Действие необнеобратимо!

1. Откройте панель исследователя
2. Нажмите "Очистить данные"
3. Подтвердите действие

Альтернативно, в консоли браузера:
```javascript
localStorage.removeItem('currentSession');
localStorage.removeItem('sessionsArchive');
```

## 📈 Расширенные возможности

### Добавление custom метрик

Отредактируйте `src/utils/sessionAnalytics.ts`:

```typescript
// Добавьте новое поле в интерфейс SessionData
export interface SessionData {
  // ... существующие поля
  customMetric: number;
}

// Добавьте метод отслеживания
trackCustomMetric(value: number) {
  if (!this.currentSession) return;
  this.currentSession.customMetric = value;
  this.saveSession();
}
```

### Интеграция с бэкендом

Для автоматической отправки данных на сервер:

```typescript
// В sessionAnalytics.ts
private async sendToServer(sessionData: SessionData) {
  try {
    await fetch('https://your-api.com/sessions', {
      method: 'POST',
      headers: { 'Content-Type': 'application/json' },
      body: JSON.stringify(sessionData)
    });
  } catch (error) {
    console.error('Ошибка отправки данных:', error);
  }
}

// Вызывайте в endSession()
endSession() {
  // ... существующий код
  this.sendToServer(this.currentSession);
}
```
