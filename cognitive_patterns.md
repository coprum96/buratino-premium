## 1. ТЕЛЕСНЫЙ БАРОМЕТР

### 🎯 Суть паттерна
Техника использования физиологических реакций тела для распознавания обмана и опасности. Основана на научном факте: тело реагирует на угрозу быстрее сознания.

### 📚 Теоретическая основа
- **Психофизиология:** Связь эмоций и телесных ощущений
- **Интуиция:** Бессознательная обработка информации
- **Соматические маркеры:** Концепция Антонио Дамасио
- **Статистика:** 87% жертв мошенничества "чувствовали что-то не то", но проигнорировали (МВД РФ)

### 🎮 Реализация в игре

#### Глава 1: Введение
**Локация:** Сцена 1.2
**Персонаж:** Сверчок Джимини (учитель)

**Диалог:**
```
"Первый инструмент - Телесный Барометр. Когда что-то не так, 
твоё тело первым знает. Почувствуешь холод, напряжение? 
Это стоп-сигнал! Тепло, лёгкость? Продолжай."
```

**Обучающая механика:**
```
КАЛИБРОВКА БАРОМЕТРА (3 шага)

Шаг 1: ВСПОМНИ ПРАВДУ
├─ Игрок вспоминает момент, когда ему говорили правду
├─ Фокус на телесных ощущениях:
│  ├─ Тепло в груди ✅
│  ├─ Лёгкость в теле ✅
│  ├─ Расслабление плеч ✅
│  └─ Спокойное дыхание ✅
└─ Задача: Запомнить это ощущение как эталон

Шаг 2: ВСПОМНИ ЛОЖЬ
├─ Игрок вспоминает момент обмана
├─ Фокус на телесных ощущениях:
│  ├─ Холод в животе 🚩
│  ├─ Сжатие в груди 🚩
│  ├─ Напряжение в плечах 🚩
│  └─ Учащённое сердцебиение 🚩
└─ Задача: Запомнить это как сигнал тревоги

Шаг 3: ИСПОЛЬЗУЙ КАК КОМПАС
├─ При новом предложении → сканируй тело
├─ Тепло/лёгкость? → Можно доверять
├─ Холод/напряжение? → СТОП, проверь!
└─ Игнорируй логические аргументы при сильном дискомфорте
```

#### Глава 5: Углублённое изучение
**Локация:** Все 4 сцены
**Персонажи:** Сверчок + Незнакомец (мошенник)

**Дополнительные аспекты:**

**1. Когда барометр НЕ работает:**
```
ЛОЖНЫЕ СИГНАЛЫ
├─ Голод → вызывает дискомфорт
├─ Усталость → искажает восприятие
├─ Раздражение → создаёт ложную тревогу
├─ Болезнь → нарушает калибровку
└─ Спешка → не даёт времени считать сигналы
```

**2. Физиология правды vs лжи:**
```
ПРАВДА (вегетативная нервная система в покое):
├─ Расслабленные мышцы
├─ Ровное дыхание
├─ Спокойный пульс
├─ Тепло в теле
├─ Открытая поза
└─ Ясность мыслей

ЛОЖЬ (активация симпатической НС):
├─ Напряжение мышц (защитная реакция)
├─ Учащённое дыхание (подготовка к борьбе/бегству)
├─ Ускоренный пульс (стресс-реакция)
├─ Холод в конечностях (кровь к органам)
├─ Закрытая поза (защита)
└─ Спутанность мыслей (кортизол)
```

**3. Практическое применение:**
В игре Буратино встречает Незнакомца, который давит на время. 
Барометр показывает:
- Холод в животе 🚩
- Напряжение в плечах 🚩
- Желание убежать 🚩

Игрок учится ДОВЕРЯТЬ этим сигналам вместо логики мошенника.

### 🎯 Игровая механика

**Интерактивный элемент:** Нет прямого UI, но:
- В диалогах с мошенниками появляются описания телесных ощущений
- Игрок делает выбор на основе этих ощущений
- Правильное использование → +мудрость
- Игнорирование сигналов → -мудрость

**Пример:**
```javascript
dialogue: {
  text: "Незнакомец давит: 'Решай сейчас!' 
         Ты чувствуешь холод в животе и напряжение.",
  choices: [
    { text: 'Игнорирую, соглашаюсь', wisdom: -20 },
    { text: 'Слушаю тело, отказываюсь', wisdom: +30 }
  ]
}
```

### 📊 Красные флажки с Телесным Барометром

**В gameData.ts реализовано:**
- 2 красных флажка в главе 1
- 4 красных флажка в главе 5
- Каждый содержит:
  - Описание игнорирования сигналов
  - Статистику жертв, которые чувствовали, но проигнорировали
  - Советы по калибровке барометра

### 💡 Обучающий эффект
**Что усваивает игрок:**
1. Тело - это детектор, а не помеха
2. Холод/напряжение = стоп-сигнал
3. Нельзя игнорировать дискомфорт
4. Калибровка улучшается с практикой

**Перенос в реальную жизнь:**
- При любом финансовом предложении - сканировать тело
- Не принимать решений при дискомфорте
- Практиковать калибровку на мелких решениях

## 2. ЗОЛОТОЙ ДЕТЕКТОР (4 КОНТРОЛЬНЫХ ВОПРОСА)

### 🎯 Суть паттерна
Система из 4 универсальных вопросов, которые выявляют 95% финансовых мошенников. Основана на психологии манипуляции и юридической защите прав потребителя.

### 📚 Теоретическая основа
- **Критическое мышление:** Метод Сократа (задавание вопросов)
- **Когнитивная психология:** Защита от эвристики доступности
- **Защита прав потребителей:** Ст. 10, 12 Закона о защите прав потребителей
- **Статистика:** 95% схем не выдерживают эти 4 вопроса (исследование ЦБ РФ)

### 🎮 Реализация в игре

#### Глава 2: Карабас и инвестпредложение
**Локация:** Сцена 2.3
**Персонаж:** Сверчок Джимини обучает, Карабас отвечает

**Структура обучения:**
```

├─ Откуда берутся деньги?
└─ Кто платит и за что?

ОТВЕТ МОШЕННИКА (Карабас):
"Это сложная финансовая схема... инвестиции... театр... билеты..."

🚩 КРАСНЫЙ ФЛАГ: Нет конкретики, только общие слова

ПРАВИЛО:
"Если не можешь объяснить бабушке - не инвестируй"
"Реальный бизнес всегда может объяснить источник дохода"

СТАТИСТИКА:
78% финансовых пирамид используют "сложные инструменты" 
как отговорку (ЦБ РФ)

ЗАКОН:
Ст. 159 УК РФ "Мошенничество": обещание без механизма

┌──────────────────────────────────────────────┐

ЧТО ПРОВЕРЯЕМ:

├─ Прописано ли в договоре?
├─ Кто несёт ответственность?
└─ Есть ли страховка/поручительство?

ОТВЕТ МОШЕННИКА:
"Конечно! Гарантирую!... То есть... риски есть везде..."

🚩 КРАСНЫЙ ФЛАГ: Гарантия испарилась при первом вопросе

ПРАВИЛО:
"Требуйте ПИСЬМЕННЫЕ гарантии в договоре"
"Высокая доходность = высокий риск ВСЕГДА"

СТАТИСТИКА:
92% мошеннических схем используют словесные "гарантии"

ЗАКОН:

о рисках в письменном виде

┌──────────────────────────────────────────────┐
│ ВОПРОС 3: Почему вы сами не вложили         │
│           все свои деньги?                   │
└──────────────────────────────────────────────┘

ЧТО ПРОВЕРЯЕМ:
├─ Логическое несоответствие
├─ Почему ищут чужие деньги?
├─ Верят ли сами в свою схему?
└─ Какая реальная история организатора?

ОТВЕТ МОШЕННИКА:
"У меня деньги заморожены в других проектах!"

🚩 КРАСНЫЙ ФЛАГ: Логическое несоответствие

АНАЛИЗ:
Если театр действительно даёт 900% за месяц, зачем Карабас:
- Ищет инвесторов среди случайных людей?
- Не закрыл "другие проекты" ради такой доходности?
- Не взял кредит в банке под 20% годовых?

ПРАВИЛО:
"Мошенники ищут чужие деньги, потому что сами не верят"
"Проверяй финансовую историю организатора"

┌──────────────────────────────────────────────┐

├─ Дают ли время на размышление?
├─ Есть ли искусственная срочность?
├─ Можно ли проконсультироваться?
└─ Право на отказ?

ОТВЕТ МОШЕННИКА:
"НЕТ! Только сегодня такие условия! Завтра будет поздно!"

🚩 КРАСНЫЙ ФЛАГ: Запрет на размышление

АНАЛИЗ:
Почему мошенники не дают времени:
- Проверка информации выявит обман
- Консультация с экспертами остановит
- Эмоции (жадность) улягутся за 24-48 часов
- Логика включится на 2-3 день

ПРАВИЛО:
"Любая срочность - красный флаг"
"Если 'только сегодня' - это 100% обман"
"Настоящие возможности подождут"

СТАТИСТИКА:
- 95% успешных схем используют давление временем (МВД РФ)
- 93% мошенников исчезают, если просишь время (МВД РФ)

ЗАКОН:
Ст. 32 ЗоЗПП: потребитель имеет право отказаться от сделки 
в течение 14 дней
```

## 3. ПРАВИЛО 72 ЧАСОВ

### 🎯 Суть паттерна
Никогда не принимать финансовых решений сразу. Давать себе минимум 3 дня (72 часа) на проверку информации и охлаждение эмоций.

### 📚 Теоретическая основа
- **Поведенческая экономика:** Охлаждающий период (cooling-off period)
- **Нейропсихология:** Эмоции vs рациональность
- **Биология решений:** Жадность и азарт живут 24-48 часов
- **Статистика:** 93% мошенников исчезают при просьбе подождать (МВД РФ)

### 🎮 Реализация в игре

#### Глава 2: Введение правила
**Локация:** Сцена 2.4
**Персонаж:** Сверчок Джимини (мудрый)

**Обучающий блок:**
```
⏰ ПРАВИЛО 72 ЧАСОВ

СУТЬ:
Никогда не принимай финансовых решений сразу.
Дай себе минимум 3 дня.

ЗАЧЕМ 72 ЧАСА?

День 1 (0-24 часа):
├─ Эмоции на пике
├─ Жадность говорит: "ХВАТАЙ!"
├─ Логика отключена
└─ Решение = импульсивное

ЗАДАЧА ДНЯ 1:
Сказать: "Я подумаю и отвечу через 3 дня"
НЕ анализировать, просто ПОДОЖДАТЬ

День 2 (24-48 часов):
├─ Эмоции начинают угасать
├─ Логика просыпается
├─ Можно начать проверку
└─ Появляются первые сомнения

ЗАДАЧА ДНЯ 2:
- Проверить компанию (реестр ЦБ, ЕГРЮЛ)
- Поискать отзывы (независимые источники)
- Посчитать реальную доходность
- Найти подвох в условиях

День 3 (48-72 часа):
├─ Эмоции улеглись
├─ Логика работает на 100%
├─ Видно то, что было скрыто
└─ Трезвое решение головой

ЗАДАЧА ДНЯ 3:
- Посоветоваться с 3 людьми, которым доверяешь
- Задать себе 4 вопроса Золотого Детектора
- Проверить Телесный Барометр
- Принять ОКОНЧАТЕЛЬНОЕ решение

ПОЧЕМУ РАБОТАЕТ:

Биология:
├─ Дофамин (жадность) падает через 24-48 часов
├─ Кортизол (стресс от срочности) нормализуется
└─ Префронтальная кора (логика) активируется

Психология:
├─ Эффект якоря ослабевает
├─ Когнитивные искажения уменьшаются
└─ Критическое мышление восстанавливается

Социология:
├─ Мошенники исчезают (93% случаев)
├─ Условия "ухудшаются" (показывая истинные)
└─ Давление перестаёт работать

ПРАВИЛО:
"Если говорят 'только сегодня' - это 100% обман"
"Настоящие возможности никуда не денутся"
"Потеря возможности < Потеря денег"
```

**Красные флажки:**
```javascript
{

  example: 'Если бы Буратино согласился сразу,

```

#### Практическое применение

**Глава 2:** Карабас давит "только сегодня!" - Буратино применяет правило
**Глава 5:** Незнакомец требует "решить за час!" - снова правило
