**Золотой Детектор** - образовательная игра для защиты от финансового мошенничества.

> ⚠️ Каждые 2.5 секунды появляется новая жертва финансового мошенничества. Не будь одной из них!

## ✨ Особенности

### Игровые возможности
- 🎭 **14 захватывающих уровней** с уникальными историями
- 💬 **Интерактивные диалоги** с выбором решений
- 📝 **Тесты и квизы** для проверки знаний
- 📚 **Образовательные материалы** встроены прямо в игру
- 🚩 **Красные флажки** - учись распознавать мошенничество
- 🎨 **Красивый современный UI** с анимациями
- 📱 **Адаптивный дизайн** - работает на всех устройствах
- ⚡ **Быстрая загрузка** - всего ~100KB gzipped

### Для исследователей
- 📊 **Автоматический сбор данных** - все действия игроков сохраняются в PostgreSQL
- 🔬 **Полная аналитика** - ответы, выборы, временные метрики
- 📈 **SQL готовые запросы** - для быстрого анализа
- 🐍 **Примеры для Python/R** - старт анализа за минуты
- 🔒 **Конфиденциальность** - данные под вашим контролем

## 🚀 Быстрый старт

### Только игра (без сбора данных)

```bash
# Установите зависимости
npm install

# Запустите dev сервер
npm run dev
```

Приложение будет доступно по адресу `http://localhost:5173`

### 🔬 Полный стек с backend и БД (для исследований)

Если вы хотите собирать данные игроков в PostgreSQL для исследования:

**📖 Полная инструкция:** [SETUP_FULL_STACK.md](./SETUP_FULL_STACK.md)

**Краткая версия:**
```bash
# 1. Создайте PostgreSQL базу
createdb buratino_research

# 2. Настройте backend
cd backend
npm install
# Создайте backend/.env (см. ENV_SETUP_INSTRUCTIONS.md)
npx prisma migrate dev
npm run dev

# 3. Настройте frontend
cd ..
# Создайте .env (см. ENV_SETUP_INSTRUCTIONS.md)
npm run dev
```

**Документация для исследователей:**
- [RESEARCH_BACKEND_README.md](./RESEARCH_BACKEND_README.md) - SQL запросы и анализ данных
- [ENV_SETUP_INSTRUCTIONS.md](./ENV_SETUP_INSTRUCTIONS.md) - Настройка переменных окружения

### Сборка для продакшна

```bash
# Соберите проект
npm run build

# Результат будет в папке dist/
```

## 🎓 Что изучают игроки

Игра обучает важным навыкам финансовой безопасности:

### Ключевые концепции
- 🔍 **Золотой Детектор** - 4 контрольных вопроса для проверки предложений
- 🚩 **Красные флажки** - признаки финансового мошенничества
- 🧡 **Телесный Барометр** - доверяй своей интуиции
- ⏰ **Правило 72 часов** - никогда не спеши с финансовыми решениями
- 🧺 **Правило 3 корзин** - распределение денег (50/30/20)

### Темы глав
1. 🪙 **Пять золотых монет** - введение в финансовую грамотность
2. 🎭 **Карабас и инвестиции** - распознавание мошенничества
3. 🧮 **Расчет реальности** - математика инвестиций
4. 🧺 **Три корзины** - категоризация инвестиций
5. 📊 **Телесный барометр** - доверие интуиции
6. 🎯 **Встреча с мошенником** - тактики манипуляции
7. 📋 **Документы и контракты** - чтение договоров
8. 🏆 **Финальное испытание** - применение всех знаний

## 🛠️ Технический стек

| Технология | Версия | Назначение |
|-----------|--------|------------|
| React | 18.2 | UI библиотека |
| TypeScript | 5.2 | Типизация |
| Zustand | 4.4 | Управление состоянием |
| Tailwind CSS | 3.4 | Стилизация |
| Vite | 5.0 | Сборщик |

### Почему эти технологии?

- **React** - популярный, производительный, большое комьюнити
- **TypeScript** - безопасность типов, автодополнение
- **Zustand** - простой и легкий store (всего 1KB!)
- **Tailwind** - быстрая разработка, консистентный дизайн
- **Vite** - мгновенный HMR, быстрая сборка
